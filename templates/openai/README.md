# DCP-AI + OpenAI Function Calling Template

Give your OpenAI function-calling agent digital citizenship in 3 steps.

## Quick Start

1. Initialize DCP identity:
   ```bash
   npx @dcp-ai/cli init
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run your agent:
   ```bash
   OPENAI_API_KEY=sk-… node index.js
   ```

## What this template does

- Registers DCP-aware tools as OpenAI functions
- Each function call creates a DCP Intent before execution
- Every response is audited in a tamper-evident chain
- A signed Citizenship Bundle is presented after the conversation

## Files

- `index.js` — OpenAI function calling with DCP integration
- `package.json` — Dependencies
- `.dcp/config.json` — DCP configuration (generated by init)
