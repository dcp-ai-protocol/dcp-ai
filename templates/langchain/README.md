# DCP-AI + LangChain Template

Get your LangChain agent digital citizenship in 3 steps.

## Quick Start

1. Initialize DCP identity:
   ```bash
   npx @dcp-ai/cli init
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run your agent:
   ```bash
   node index.js
   ```

## What this template does

- Creates a LangChain agent with DCP digital citizenship
- Every action is declared as an Intent
- Every response is audited
- The agent carries a verifiable Citizenship Bundle

## Files

- `index.js` — LangChain agent with DCP integration
- `.dcp/config.json` — DCP configuration (generated by init)
- `package.json` — Dependencies
