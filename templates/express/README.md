# DCP-AI + Express API Server Template

Build an API that verifies AI agent citizenship in 3 steps.

## Quick Start

1. Initialize DCP identity:
   ```bash
   npx @dcp-ai/cli init
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   node index.js
   ```

## What this template does

- Runs an Express API server with DCP verification middleware
- Agents must present a signed Citizenship Bundle to access protected routes
- Bundles are verified for structural integrity, session binding, and signatures
- Provides `/verify`, `/health`, and `/agents` endpoints

## Endpoints

| Method | Path               | Auth     | Description                        |
|--------|--------------------|----------|------------------------------------|
| GET    | `/health`          | None     | Health check                       |
| POST   | `/verify`          | None     | Verify a DCP bundle (standalone)   |
| POST   | `/agents/action`   | DCP      | Perform agent action (requires bundle) |
| GET    | `/agents/registry` | DCP      | List registered agents             |

## Files

- `index.js` — Express server with DCP middleware
- `package.json` — Dependencies
- `.dcp/config.json` — DCP configuration (generated by init)
